## 本地计算

### 使用 Github 前端

```
cd core && python runner.py
```

### 使用本地前端

```
cd core && python runner.py -local
```

## 网页前端

### 安装依赖

```
npm install
```

### 本地调试

```
npm run serve
```

### 本地构建

```
npm run build-local
```

### Github Action 构建

```
npm run build-action
```
